# **NarrativeX AI**

**All-in-One AI Storytelling Tool**

NarrativeX AI is an AI-powered storytelling platform that allows users to create rich, immersive stories, detailed characters, structured plots, and fictional worlds. The tool also analyzes stories to provide feedback on structure, pacing, and consistency.

## **Features**

* **Story Creator** – Generate complete stories based on genre, theme, characters, and settings
* **Character Creator** – Build detailed characters with personality, backstory, and traits
* **Plot Generator** – Automatically create engaging story arcs and subplots
* **World Builder** – Construct fictional worlds with culture, geography, and lore
* **Story Analyzer** – Evaluate stories for structure, pacing, character consistency, and suggestions

## **Installation**

**1. Clone the repository**

```bash
git clone https://github.com/yourusername/narrativex-ai.git
cd narrativex-ai
```

**2. Setup backend (Flask + Python)**

```bash
cd backend
python -m venv venv
source venv/bin/activate  # Windows: venv\Scripts\activate
pip install -r requirements.txt
```

**3. Setup frontend (React.js)**

```bash
cd frontend
npm install
npm start
```

**4. Configure OpenAI API Key**
Create a `.env` file in the backend folder:

```env
OPENAI_API_KEY=your_openai_api_key_here
```

**5. Run the application**

* Backend: `python app.py`
* Frontend: `npm start` (usually runs on `http://localhost:3000`)

## **Usage**

1. Open the app in your browser
2. Select the feature you want to use:

   * Story Creator
   * Character Creator
   * Plot Generator
   * World Builder
   * Story Analyzer
3. Enter the required inputs (genre, characters, prompts, etc.)
4. Click **Generate**
5. View the generated output, copy it, or refine using the tool
